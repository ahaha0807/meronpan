# API
## 情報一覧取得
### 概要
- トップページ用の情報を取得するためのAPI
- 最大取得（表示）件数は20件

### URL
`GET /api/v1/meronpans`

### クエリなど


## 詳細情報取得
### 概要
- 選択したメロンパンの情報を取得するためのAPI
- 情報一つ一つに付与された（kintoneから受け取った）コードを元にアクセスする

### URL
`GET /api/v1/meronpan/:id`

### クエリなど
- `:id` -> 数字○文字

## 情報登録
### 概要
- メロンパンの情報を新しく登録するためのAPI

### URL
`POST /api/v1/meronpan`
